# TripSync Backend

TripSync is a travel planner app designed to help users organize and manage trips efficiently. This repository contains the backend of TripSync, built with **Django**, **Django REST Framework**, and **Simple JWT** for authentication. It provides secure and scalable APIs for the Android frontend.

## 🚀 Features

- 🔐 **JWT Authentication** using Django REST Framework and Simple JWT
- 🧾 User Registration & Login
- 🔄 Token Refresh Support
- 🔒 Protected Endpoints for Authenticated Users
- 🌍 Trip and itinerary planning APIs *(coming soon)*

## 🛠️ Tech Stack

- Python 3.x
- Django 4.x
- Django REST Framework
- Simple JWT
